# SVM Kernel Tutorial

This repository demonstrates how linear, polynomial, and RBF kernels shape SVM decision boundaries.

## Files

- `tutorial.pdf` – The main tutorial with explanations and figures.
- `SVM_kernels_notebook.ipynb` – Jupyter Notebook with code to reproduce figures.
- `figures/` – Optional folder with saved PNG plots.

## Instructions

1. Open `SVM_kernels_notebook.ipynb` in Google Colab or Jupyter Notebook.
2. Run all cells to reproduce all plots.
3. Figures are automatically saved in the `figures/` folder.
4. Parameters (C, gamma, degree) are annotated in code.
